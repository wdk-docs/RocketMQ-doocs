故障处理
=============


使用异常
----------------------

- 无法连接 Broker
- 启动 Producer、Consumer 失败，Group ID 重复
- 广播模式下，消费者启动加载 JSON 文件异常
- 主动订阅消息，获取队列列表失败
- 消息队列 RocketMQ 报错 “Can not find name server”
- 消息显示 _Consumed_, 但消费端未感知到

资源不存在
----------------------

- Group ID 不存在主机名不存在

状态不一致
----------------------

- 非预期的客户端连接消息不合法参数不合法客户端状态异常订阅关系不一致

----------------------
